# Pilot – AI Support Studio (MVP)

This is a minimal full-stack MVP for **Pilot**, a SaaS dashboard for managing AI support agents.

## Tech stack

- Frontend: React + TypeScript + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: Supabase (PostgreSQL) – schema provided, mocked in code
- Deployment target: Vercel (frontend), any Node host for backend

## Getting started

### 1. Backend

```bash
cd backend
npm install
npm run dev
